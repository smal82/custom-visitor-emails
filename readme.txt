Registra e visualizza le visite dei visitatori con data, ora, IP e User-Agent. 

== Description ==

Il plugin "Custom Visitor Emails" ti consente di registrare e visualizzare le visite dei visitatori sul tuo sito. Ogni volta che un utente, escluso l'admin, accede al sito, il plugin registra automaticamente la data, l'ora, l'indirizzo IP e lo User-Agent della visita. Le visite vengono inviate per email all'admin e visualizzate in una tabella dedicata nella pagina del plugin con un menu dedicato, insieme ai dettagli di data, ora, indirizzo IP e User-Agent. Il plugin offre la possibilità di svuotare la tabella delle visite registrate, fornendo un'opzione di gestione semplice e comoda. Inoltre il plugin consente di modificare i files robots.txt e .htaccess per poter così bloccare tutte le visite che l'admin ritiene inopportune.

Un'interfaccia intuitiva rende il plugin "Custom Visitor Emails" uno strumento utile per monitorare l'attività dei visitatori sul tuo sito WordPress.
